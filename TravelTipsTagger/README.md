# 旅游攻略标签提取器

## 当前版本
**V1.1.0**

## 工具介绍
一个针对长文本旅游攻略进行关键信息提取的文本挖掘工具。

## 功能实现
1.  **旅游攻略相关地区信息提取**：从保存在excel文件中的旅游攻略文本中提取出攻略所针对的城市名称和其在文章中的出现次数
